## üöÄ Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - runs entirely client-side

### Installation
1. Clone the repository
```bash
git clone https://github.com/yourusername/TeamSynapse.git
cd TeamSynapse
```

2. Open `index.html` in your browser
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

Or simply double-click `index.html` to open in your default browser.

## üé® Customization

### Updating Team Members
Edit `index.html` and find the team section. Replace placeholder content:
```html
<h3 class="member-name">Your Name</h3>
<p class="member-role">Your Role</p>
```

### Updating Projects
Edit project cards in `index.html`:
```html
<h3>Your Project Name</h3>
<p>Your project description...</p>
```

### Color Scheme
Modify CSS variables in `styles.css`:
```css
:root {
    --accent-blue: #00d4ff;
    --accent-purple: #8b5cf6;
    --accent-cyan: #06b6d4;
}
```

### Contact Information
Update email and social links in `index.html`:
```html
<p>your-email@example.com</p>
<a href="https://github.com/yourusername">GitHub</a>
```
